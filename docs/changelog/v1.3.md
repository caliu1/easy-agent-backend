# v1.3 更新日志

更新时间：2026-04-21

## 概览

v1.3 聚焦平台“可用性与运营闭环”能力，新增了两条核心主线：

1. 用户注册登录与账号落库（摆脱单一 `admin` 演示账号）
2. Agent 广场订阅体系（订阅关系持久化 + 我的订阅视图）

---

## 核心更新

### 1) 用户认证体系上线（注册 / 登录 / 落库）

新增能力：

- 注册接口：`POST /api/v1/user_register`
- 登录接口：`POST /api/v1/user_login`
- 账号持久化表：`ai_user_account`
- 登录成功更新最近登录时间
- 前端登录页支持“登录 / 注册”切换，不再硬编码 `admin/admin`

涉及模块：

- `easy-agent-api`：`IUserAuthService`、认证 DTO
- `easy-agent-domain`：`IUserAuthManageService`、密码校验与加密
- `easy-agent-infrastructure`：`UserAccountRepository`、DAO、PO
- `easy-agent-trigger`：`UserAuthController`
- `easy-agent-front`：`/login` 页改造 + 认证 API 接入

---

### 2) Agent 广场订阅能力

新增能力：

- 订阅接口：`POST /api/v1/agent_config_subscribe`
- 取消订阅：`POST /api/v1/agent_config_unsubscribe`
- 我的订阅列表：`GET /api/v1/agent_config_my_subscribe_list`
- 订阅关系持久化表：`ai_agent_subscribe`
- 仅允许订阅当前“广场可见”的 Agent

前端体验：

- 广场卡片右下角爱心按钮
- 未订阅：空心爱心
- 已订阅：实心爱心
- 首页新增“我的订阅”分栏，订阅后可立即查看

涉及模块：

- `easy-agent-api`：订阅请求 DTO 与服务接口补充
- `easy-agent-domain`：订阅查询/订阅/取消订阅领域服务
- `easy-agent-infrastructure`：订阅 DAO/PO/Repository
- `easy-agent-trigger`：订阅相关 Controller 接口
- `easy-agent-front`：首页分栏与卡片订阅交互
